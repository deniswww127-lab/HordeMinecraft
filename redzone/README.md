# ☢️ ПРОЕКТ: КРАСНАЯ ЗОНА (Project: RED ZONE)

Серверная часть hardcore PvPvE extraction-хоррора на Minecraft **1.20.4**
(Paper/Spigot). Игроки заходят по IP без модов и лаунчеров; вся механика — на
плагинах и скриптах.

> **Важно про репозиторий.** Этот git-репозиторий исторически содержит
> отдельный Python-проект «ПостПилот ВК» (папка `app/`, `static/`, `index.html`).
> Он к Minecraft отношения не имеет. Весь код «Красной Зоны» лежит **только**
> в папке `redzone/` и не пересекается с ним.

## ✅ Этап 1 — Система веса + хардкор-медицина (готово к тесту)

Реализовано на **Denizen**. Файлы — в `redzone/Denizen/scripts/`.

### Что делает
- **Вес груза.** Каждый предмет имеет вес. Суммарный вес инвентаря считается
  раз в секунду:
  - `< 25 кг` — полная подвижность (можно бежать);
  - `25–35 кг` — нельзя бежать (только шаг);
  - `> 35 кг` — еле идёт (скорость снижена) + нельзя бежать.
- **Хардкор-медицина.** Здоровье **не восстанавливается само** (отключена
  регенерация от сытости).
  - **Кровотечение** — с шансом открывается при пулевом/рукопашном ранении,
    снимает по ½ сердца каждые 4 сек. Лечится **бинтом**.
  - **Перелом ноги** — при сильном падении. Заставляет хромать (как перегруз)
    и не даёт бежать. Лечится **шиной**.
- **Медикаменты** (`Бинт`, `Шина`, `Аптечка`) применяются по ПКМ с «кастом»
  (3–5 сек). Полученный урон **прерывает** применение — предмет не тратится.
- **HUD** на босс-баре: `HP` + `Груз` + индикаторы `[КРОВЬ]` / `[НОГА]`.

### Файлы модуля
```
redzone/Denizen/scripts/
  rz_core.dsc       Конфиг, отключение саморегена, сброс травм при возрождении
  rz_weight.dsc     Подсчёт веса, штрафы передвижения, босс-бар HUD
  rz_medicine.dsc   Кровотечение и переломы (триггеры + тик урона)
  rz_items.dsc      Item-скрипты: Бинт, Шина, Аптечка
  rz_use.dsc        Применение медикаментов (ПКМ + каст с прерыванием)
  rz_admin.dsc      Отладочная команда /rz
```

## 🚀 Установка на сервер

1. Скопируй все файлы из `redzone/Denizen/scripts/` в папку плагина на хостинге:
   ```
   plugins/Denizen/scripts/
   ```
   (можно положить в подпапку — Denizen грузит скрипты рекурсивно).
2. Перезагрузи скрипты в игре или из консоли:
   ```
   /denizen reload scripts
   ```
   либо просто перезапусти сервер.
3. Проверь консоль — должно появиться:
   `[RED ZONE] Конфигурация модуля выживания загружена.`
4. Выдай себе право на отладку (или просто будь OP):
   ```
   /lp user <ник> permission set rz.admin true
   ```

> Denizen должен быть настроен на загрузку скриптов (по умолчанию включено).
> Если скрипты не грузятся — смотри `/denizen debug` и логи на ошибки парсинга.

## 🧪 Как протестировать (5 минут)

1. `/rz give all` — получишь 3 бинта, 2 шины, 1 аптечку.
2. Над хотбаром появится **босс-бар** с HP и грузом.
3. **Вес:** набей инвентарь блоками (`/give` камня/железа стопками) — на ~25 кг
   перестанешь бегать, на ~35 кг начнёшь еле идти. `/rz weight` покажет цифру.
4. **Кровотечение:** `/rz bleed` — пойдёт урон по ½ сердца, в HUD загорится
   `[КРОВЬ]`. ПКМ бинтом (3 сек, не двигайся и не получай урон) — остановится.
5. **Перелом:** `/rz fracture` (или спрыгни с высоты) — захромаешь, в HUD
   `[НОГА]`. ПКМ шиной (4 сек) — пройдёт.
6. **Аптечка:** получи урон, затем ПКМ аптечкой (5 сек) — +4 сердца.
7. **Прерывание:** начни перевязку и получи удар — каст сорвётся, бинт цел.
8. `/rz heal` — снять всё; `/rz setup` — перечитать конфиг после правок.

## ⚙️ Настройка баланса

Значения правятся в `rz_core.dsc` (задача `rz_setup`), затем `/rz setup`:

| Параметр | Значение | Что делает |
|---|---|---|
| `rz_light_max` | 25 | Порог веса, выше — нельзя бежать |
| `rz_heavy_max` | 35 | Порог веса, выше — еле идёт |
| `rz_bleed_chance` | 40 | Шанс кровотечения с попадания, % |
| `rz_fracture_min_dmg` | 4 | Урон падения для перелома, HP |
| `rz_weights` | map | Таблица весов ванильных материалов |

Вес кастомных предметов задаётся флагом `rz_weight` прямо в item-скрипте.

## ⚠️ Технические решения и ограничения (этап 1)

- **HUD на босс-баре, а не на actionbar.** Actionbar в рейде займёт
  WeaponMechanics (патроны). Чтобы не мерцало — survival-HUD пока на босс-баре.
  Финальный «immersive» HUD сделаем ресурс-паком позже.
- **Запрет бега = шкала голода 6.** Спринт в Denizen нельзя отменить событием,
  поэтому при перегрузе/переломе голод занижается до 6 (Minecraft сам блокирует
  бег). Шкалу голода спрячем ресурс-паком. Поэтому **голод как механику пока не
  используем** (саморегенерация и так отключена).
- **Носимая броня** тоже попадает в вес (реалистично, но можно отключить).
- **Дроп предмета во время каста** — теоретически позволяет применить эффект,
  не потратив медикамент. Закроем в следующей итерации, если будет критично.
