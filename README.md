# ПостПилот — генерация и публикация постов в сообществах ВКонтакте

Сервис, который подключается к сообществу ВК, генерирует пост по теме/тону/формату
и публикует его от имени сообщества. При подключении пользователь выбирает,
в каких именно сообществах сервис будет работать.

## Что уже работает
- Вход через VK OAuth (scope `groups`)
- Получение списка сообществ, где пользователь — администратор
- Экран выбора сообществ
- Получение токенов выбранных сообществ (scope `manage,wall,photos`)
- Генерация текста поста через LLM (Anthropic) с шаблонным фолбэком
- Публикация на стену сообщества (`wall.post`, `from_group=1`)
- Сохранение пользователей / сообществ / постов в БД
- Заготовка отложенной публикации (статус `queued` + `publish_at`)

## Структура
```
app/
  config.py            настройки из .env
  db.py                подключение к БД
  models.py            User, Community, Post
  services/
    vk.py              OAuth, список групп, публикация
    generator.py       генерация текста (LLM + фолбэк)
  routers/
    auth.py            /auth/vk/* — вход и подключение сообществ
    posts.py           /api/* — генерация и публикация
  main.py              точка входа
static/index.html      фронтенд (подключение → выбор → генератор)
```

## Запуск локально
```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env        # заполнить VK_CLIENT_ID / VK_CLIENT_SECRET
uvicorn app.main:app --reload
```
Открыть http://localhost:8000

Без VK-ключей работает только UI и генерация (фолбэк). Для реальной публикации
нужны ключи приложения VK.

## Что нужно сделать в VK перед выпуском
1. Создать приложение на https://dev.vk.com → тип **Веб-сайт**.
2. Скопировать `client_id` и `client_secret` в `.env`.
3. Указать **Redirect URI** = `https://ваш-домен/auth/vk/callback`
   (точно совпадать с `VK_REDIRECT_URI`).
4. Развернуть сервис на домене с HTTPS (VK требует https для OAuth).
5. Отправить приложение на модерацию VK.

## Поток подключения сообщества
1. Пользователь жмёт «Войти через ВКонтакте» → `/auth/vk/login`.
2. VK возвращает `code` на `/auth/vk/callback` → меняем на токен пользователя.
3. `groups.get(filter=admin)` → список сообществ → редирект на `/?select=...`.
4. Пользователь отмечает сообщества → `POST /auth/vk/select`.
5. Редирект в VK за токенами сообществ (scope `manage,wall,photos`).
6. Токены сохраняются → можно публиковать.
